## 중간고사 시험 정리
```
시작 터미널 명령어: ./mvnw quarkus:dev

<!DOCTYPE>: 문서 형식 정의. 
<html>>: 모든 HTML요소의 최상위 컨테이너. 문서의 시작과 끝

<head>: 브라우저 화면에 보이지 않는 설정 값을 담음

<title>: 브라우저 상단 탭에 표시되는 제목

<body>: 사용자에게 실제로 보이는 모든 콘텐츠 작성

<div>: 레이아웃을 나누는 박스 모델

백엔드-웹3.0의 현실화(분산 블록체인)

하이퍼 링크: <a href=" "> 페이지 연결할 때 사용</a>

이미지 로딩: <img src=" ">

../: 한 단계 상위 폴더 
```
